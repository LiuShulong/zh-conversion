zh-conversion
=====

中文简体和繁体转换工具,数据字体对照表来自于 MediaWiki

```php
use slliu\zh\Convert;
$sRet = Convert::zhconversion_hant($sChar);
```

Install
-------

Install `zh-conversion` using Composer.

```
$ composer require slliu/zh-conversion
```

Test
-------

```
composer run-script test
```
